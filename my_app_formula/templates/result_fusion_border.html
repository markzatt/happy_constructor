Согласно пункту 14.1.16 СП 16.13330.2011 выбор сечения для расчета осуществляется по формулам: \({B_f R_{wf}\over B_z R_{wz}}\leqslant 1 \) - по металлу шва или \({B_f R_{wf}\over B_z R_{wz}} > 1 \) - по границе сплавления.

На основании исходных данных, расчет должен быть выполнен по границе сплавления.

Таким образом согласно пункта 14.1.17 СП 16.13330.2011 расчет сварных соединений с угловыми швами при действии момента \(М\) в плоскости, перпендикулярной плоскости расположения швов, следует выполнять на срез (условный) по границе сплавления по формуле \(M / (W_ { z } R_ { wz } \gamma _ { c }) \leqslant 1 \).
